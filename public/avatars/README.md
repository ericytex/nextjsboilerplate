# Default Avatar

Place the default user avatar image here as `default-avatar.png`.

The image should be:
- **Format**: PNG (recommended) or JPG
- **Size**: 400x400px (recommended)
- **File name**: `default-avatar.png`
- **Location**: `/public/avatars/default-avatar.png`

This image will be used as the default avatar for all users who haven't uploaded a custom profile picture.

## Current Image

The default avatar is a stylized futuristic illustration of a person with headphones in neon blue/purple colors, perfect for a tech/AI-themed application.
